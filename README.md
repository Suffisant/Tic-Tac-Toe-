Python Tic-Tac-Toe

A terminal-based implementation of the classic Tic-Tac-Toe (Noughts and Crosses) game. This project features multiple versions of the game, allowing for local multiplayer or challenging a computer opponent.
 
🔍 Overview
This project is built using pure Python and demonstrates core programming concepts such as multidimensional arrays (for the board), game loops, and basic AI logic for computer moves. It provides a lightweight way to play the game directly in your command line.

🎮 Game Modes
The repository contains three different scripts tailored to your preferred playstyle:

Human vs. Human: Local 1v1 mode where two players take turns on the same machine.

Human vs. Computer: Play against the computer, with the human taking the first move.

Computer vs. Human: Challenge yourself against a computer that starts the game.

✨ Features
Dynamic Board Display: The 3x3 grid updates in the terminal after every move.

Input Validation: Prevents players from choosing occupied cells or entering invalid coordinates.

Automated Win Detection: Instantly recognizes wins across rows, columns, and diagonals, or identifies a draw.

Computer Logic: Features basic automated move selection for the computer opponent.

🕹 How to Play
The game is played on a 3x3 grid.

Players are assigned X or O.

On your turn, enter the number or coordinates corresponding to the square where you want to place your mark.

The first player to get 3 of their marks in a row (up, down, across, or diagonally) is the winner.

If all 9 squares are full and no player has 3 marks in a row, the game ends in a draw.
